Crux timing test

This test runs the Crux search engines, Tide and Comet, on a fixed set of spectra and a fixed database, comparing the running time in a variety of parameter settings. The parameters that are varied include

  • precursor m/z (3 Th or 10 ppm),
  • fragment bin size (1 Th or 0.02 Th),
  • number of threads (1 or 4), and
  • (for Tide) score function (XCorr, exact p-value, or combined p-value).

For Tide, we use spectra that have been converted into the spectrumrecords format, and the database has been indexed by tide-index.

Wall clock time is measured by Crux internally.
