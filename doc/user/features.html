Features used by percolator and q-ranker

Following is a list of the features used by the crux percolator and crux q-ranker commands to represent each peptide-spectrum match (PSM).

Number Name Type Description Percolator q-ranker barista
1 XCorr Score Float The SEQUEST cross-correlation score ✓ ✓ ✓
2 DeltCN Float The difference between this PSM's XCorr and the XCorr of the second-ranked PSM for this spectrum, divided by this PSM's XCorr. Note that this definition differs from that of the standard delta Cn reported by SEQUEST. ✓ ✓ ✓
3 Sp score Float The preliminary SEQUEST score. ✓ ✓ ✓
4 lnrSp Float The natural logarithm of the rank of the match based on the Sp score ✓
5 Mass Float The observed mass [M+H]+ ✓ ✓ ✓
6 dM Float The difference in calculated and observed mass ✓    
7 absdM Float The absolute value of the difference in calculated and observed mass ✓    
8 ionFrac Float The fraction of matched b and y ions ✓
9 lnSM Float The natural logarithm of the number of database peptides within the specified m/z range ✓
10 enzN Boolean Is the peptide preceded by an enzymatic (tryptic) site? ✓ ✓ ✓
11 enzC Boolean Does the peptide have an enzymatic (tryptic) C-terminus? ✓ ✓ ✓
12 enzInt Integer Number of missed internal enzymatic (tryptic) sites ✓
13 pepLen Integer The length of the matched peptide, in residues ✓ ✓ ✓
14-16 charge 1-3 Boolean Is this a 1+/2+/3+ charged spectrum? ✓ ✓ ✓
17 scan Integer The identifying number of each scan.   ✓ ✓
18 label Boolean Is this decoy or target?   ✓ ✓
19 xcorr rank Integer The rank of this PSM when sorted by xcorr   ✓ ✓
20 Sp rank Integer The rank of this PSM when sorted by sp score.   ✓ ✓
21 t-term enz Boolean Does the peptide have an enzymatic(tryptic) T_terminus.   ✓ ✓
22 missed cleavage The cleavage rules for generating this peptide based on the user-specified enzyme specificity.   ✓ ✓
23 number of sequnce_comparision Integer The number of sequnce comparision   ✓ ✓
24 delta mass Float Difference between measured and calculated mass   ✓ ✓
25 abs(delta mass) Float Absolute value of delta mass   ✓ ✓
26 b ions Float The mass of b-ions which extended from N_Terminus.   ✓ ✓
27 y ions Flaot The mass of y ions which extended from C_Terminus.   ✓ ✓
28 flanking Float   ✓ ✓
