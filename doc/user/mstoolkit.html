MSToolkit

MSToolkit is a library for handling mass spectrometry data in various file formats. Crux is capable of using MSToolkit to parse spectrum files. The ProteoWizard parser is used by default, but the MSToolkit parser can be used by commands that support the spectrum-parser parameter.

The following is a list of spectrum file formats supported by MSToolkit.

  • MS1/CMS1/BMS1
  • MS2/CMS2/BMS2
  • ZS
  • UZS
  • MSMAT
  • RAW
  • MZXML[.GZ]
  • MZ5
  • MZML[.GZ]
  • MGF (Mascot Generic Format)
  • SQLITE3
  • PSM
